# News Summarizer Extension

This Chrome extension provides concise summaries of news articles using advanced natural language processing powered by the Cohere AI API.

## Features

- Summarizes news articles from The Washington Post and The Wall Street Journal
- Displays summaries directly on the webpage
- Customizable API key for Cohere AI integration

## Installation

1. Clone this repository: git clone https://github.com/ChaithanyaSaiB/News-Summarizer-Extension.git
2. Navigate to `chrome://extensions/` in your Chrome browser
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked" and select the cloned repository folder

## Usage

1. Click the extension icon in your browser toolbar
2. Enter your Cohere AI API key in the options page
3. Navigate to a news article on The Washington Post and The Wall Street Journal websites
4. The extension will automatically generate and display a summary underneath the article heading

## Configuration

To set up your Cohere AI API key:

1. Click the extension icon and select "Options"
2. Enter your API key in the provided field
3. Click "Save"

## Development

This project uses JavaScript and the Chrome extension API. To modify the extension:

1. Edit the relevant files in the project directory
2. Reload the extension in `chrome://extensions/`
